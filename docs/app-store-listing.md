## 构建上传 (Build)

ASC blocker 之一是 "必须选择一个构建版本"。需要：

```bash
cd app-rn
eas build --profile production --platform ios
```

15-20 min 后 EAS 自动 upload 到 ASC。回 ASC iOS App 版本 → 构建版本 → "+" 选刚 upload 的。
